Name Description Example
echo echo the STRING or STRINGS to the standard output sp2a@rpi04:~$ echo yeet "yeet"
ls The Ls function will list the contents of the directory. Inforation about the files in the directory will be listed sp2a@rpi04:~$ ls "scripts readme website"
ps The ps function lists the current running processes and their PIDs sp2a@rpi04:~$ ps -s "824"
kill This functon terminates the running processes sp2a@rpi04:~$ kill -62853
hostname Obtains the Domain Name System for the computer sp2a@rpi04:~$ hostname -myComputers
sleep Creates a dummy job that is able to delay the execution of certain programs sp2a@rpi04:~$ sleep 500
cd This command changes the current working directory sp2a@rpi04:~$ cd "directory 04"
cp This command copies the Source File to the Destination file of the Directory sp2a@rpi04:~$ cp "cp a.txt b.txt"
date This command displays the current date and time of the Linux operating system sp2a@rpi04:~$ date "Fri May 17 20:00:00 PDT 2019"
uname -a This command reports the basic information about a computer's software and hardware sp2a@rpi04:~$ uname -o "system of operation"
